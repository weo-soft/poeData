Why Your Contributions Matter

To accurately calculate item weightings and probabilities, we need as many datasets as possible. Each dataset you contribute helps improve the accuracy of weight calculations for the entire community.

More datasets = More accurate calculations

How to collect data

The most basic information that are needed in the dataset are the items that were used as input and the items that were received as output.

In general you should only use one item as an input per dataset, different inputs should be treated as different datasets.

There are two main approaches to collect data on the weight of the items:

  • Using a Vendor recipe (3-to-1)
  • Rerolling the item using the Harvest Bench.

The Vendor recipe in general accepts any combination of items of the same type, though you should only use the same item as input per dataset to recieve accurate results.

The same approach, of only using one item as an input per dataset, applies to the harvest bench as well.

When using the Harvest Bench to reroll items it is recommended to use single items to reroll, since using stacks does not provide any additional information about the weight, but increases the costs per transaction while reducing the number of transactions for the same amount of items.

Note: There are certain item categories that have slightly different or additional approaches on collecting data, notably Scarabs, Divination Cards and Contracts, the Details are located in the respective category pages.

Submitting your data

When submitting your dataset, you should include the following information:

  • Input Items: The items you used as input
  • Items: The output items with their counts
  • Description: A short description of how the data was obtained
  • Date: The date when the data was collected (format: YYYY-MM-DD)
  • Patch: (Optional) The game patch version (e.g., "3.26.0")

If you made a post, eg. on reddit or discord, you can also include the link to the post in the submission.

Dataset size recommendations

Since we are trying to calculate/estimate probabilities we need datasets with as many transactions as possible. The more transactions you have, the more accurate the results will be.

It is recommend to collect at least 1000+ input items for basic results, and 10000+ for better accuracy.

How to Submit

Datasets can be submitted by using the dataset submission interface to either using the form to directly fill in your data or by providing your data via a file or link.
