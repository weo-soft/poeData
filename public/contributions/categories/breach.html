There is one approach to collect data on weightings:

  • Rerolling Breach items using the Harvest Bench

Harvest Bench Reroll

When using the Harvest Bench to reroll Breach items, it is important to use the same Breach item as input for the trades to not skew the result. When using different Breach items as inputs, separate the results and treat them as different Datasets:

  1. Choose Your Input Breach Item: Select a specific Breach item type to use as input (e.g., Chayula's Breachstone, Splinter of Chayula)
  2. Harvest Bench Reroll: Use the Harvest Bench to reroll a single Breach item at a time
  3. Batch Collection:
    • Reroll items one at a time (do not use stacks, as this doesn't provide additional information but increases costs)
    • Record each reroll result individually
    • Repeat this process as many times as you can
  4. Record Results: Note exactly which Breach item you used as input and all the Breach items you received back, including their counts
  5. Continue Collecting: The more rerolls you complete, the more accurate your dataset will be

Note: It is recommended to collect at least 1000+ input items for basic results, and 10000+ for better accuracy.
