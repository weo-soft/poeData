There is one approach to collect data on weightings:

  • Using a Vendor recipe (3-to-1)

3-to-1 Vendor recipe

When using the Vendor recipe, it is important to use the same Tattoo as input for the trades to not scew the result. When using different tattoos as inputs, separate the results and treat them as different Datasets:

  1. Choose Your Input Tattoo: Select a specific tattoo type to use as input (e.g., Tattoo of the Arohongui Moonwarden, Tattoo of the Rongokurai Turtle)
  2. Vendor Recipe: Sell 3 tattoos of that type to a vendor to receive 1 random tattoo back
  3. Batch Collection:
    • Since you can only fit 60 items in your inventory, and for efficiency, work in batches of 180 tattoos (18 stacks of 10)
    • Sell 180 input tattoos (18 Ã— 10) to get 60 random tattoos back
    • Repeat this process as many times as you can
  4. Record Results: Note exactly which tattoo you used as input and all the tattoos you received back, including their counts
  5. Continue Collecting: The more batches you complete, the more accurate your dataset will be

Note: Journey Tattoos can not be rerolled, neither as input nor as output.
