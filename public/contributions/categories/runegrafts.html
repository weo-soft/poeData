There is one approach to collect data on weightings:

  • Using a Vendor recipe (3-to-1)

3-to-1 Vendor recipe

When using the Vendor recipe, it is important to use the same Runegraft as input for the trades to not skew the result. When using different runegrafts as inputs, separate the results and treat them as different Datasets:

  1. Choose Your Input Runegraft: Select a specific runegraft type to use as input
  2. Vendor Recipe: Sell 3 runegrafts of that type to a vendor to receive 1 random runegraft back
  3. Batch Collection:
    • Since you can only fit 60 items in your inventory, and for efficiency, work in batches of 180 runegrafts (18 stacks of 10)
    • Sell 180 input runegrafts (18 Ã— 10) to get 60 random runegrafts back
    • Repeat this process as many times as you can
  4. Record Results: Note exactly which runegraft you used as input and all the runegrafts you received back, including their counts
  5. Continue Collecting: The more batches you complete, the more accurate your dataset will be
