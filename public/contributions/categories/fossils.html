For general information on contributing datasets, please refer to the Contribution page.

There is one approach to collect data on weightings:

  • Rerolling Fossils using the Harvest Bench

Harvest Bench Reroll

When using the Harvest Bench to reroll Fossils, it is important to use the same Fossil as input for the trades to not skew the result. When using different Fossils as inputs, separate the results and treat them as different Datasets:

  1. Choose Your Input Fossil: Select a specific Fossil type to use as input (e.g., Dense Fossil, Bound Fossil)
  2. Harvest Bench Reroll: Use the Harvest Bench to reroll a single Fossil at a time
  3. Batch Collection:
    • Reroll items one at a time (do not use stacks, as this doesn't provide additional information but increases costs)
    • Record each reroll result individually
    • Repeat this process as many times as you can
  4. Record Results: Note exactly which Fossil you used as input and all the Fossils you received back, including their counts
  5. Continue Collecting: The more rerolls you complete, the more accurate your dataset will be

Note: It is recommended to collect at least 1000+ input items for basic results, and 10000+ for better accuracy.
