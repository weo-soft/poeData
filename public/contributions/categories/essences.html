There is one approach to collect data on weightings:

  • Rerolling Essences using the Harvest Bench

Harvest Bench Reroll

When using the Harvest Bench to reroll Essences, it is important to use the same Essence as input for the trades to not skew the result. When using different Essences as inputs, separate the results and treat them as different Datasets:

  1. Choose Your Input Essence: Select a specific Essence type to use as input (e.g., Essence of Greed, Essence of Contempt)
  2. Harvest Bench Reroll: Use the Harvest Bench to reroll a single Essence at a time
  3. Batch Collection:
    • Reroll items one at a time (do not use stacks, as this doesn't provide additional information but increases costs)
    • Record each reroll result individually
    • Repeat this process as many times as you can
  4. Record Results: Note exactly which Essence you used as input and all the Essences you received back, including their counts
  5. Continue Collecting: The more rerolls you complete, the more accurate your dataset will be

Note: It is recommended to collect at least 1000+ input items for basic results, and 10000+ for better accuracy.
