There is one approach to collect data on weightings:

  • Rerolling Delirium Orbs using the Harvest Bench

Harvest Bench Reroll

When using the Harvest Bench to reroll Delirium Orbs, it is important to use the same Delirium Orb as input for the trades to not skew the result. When using different Delirium Orbs as inputs, separate the results and treat them as different Datasets:

  1. Choose Your Input Delirium Orb: Select a specific Delirium Orb type to use as input (e.g., Delirium Orb of the Skittering, Delirium Orb of the Apocalypse)
  2. Harvest Bench Reroll: Use the Harvest Bench to reroll a single Delirium Orb at a time
  3. Batch Collection:
    • Reroll items one at a time (do not use stacks, as this doesn't provide additional information but increases costs)
    • Record each reroll result individually
    • Repeat this process as many times as you can
  4. Record Results: Note exactly which Delirium Orb you used as input and all the Delirium Orbs you received back, including their counts
  5. Continue Collecting: The more rerolls you complete, the more accurate your dataset will be

Note: It is recommended to collect at least 1000+ input items for basic results, and 10000+ for better accuracy.
