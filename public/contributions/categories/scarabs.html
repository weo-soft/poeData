Overview

For general information on contributing datasets, please refer to the Contribution page.

There are two approaches to collect data on weightings:

  • Using a Vendor recipe (3-to-1)
  • Opening veiled Scarabs (Only available in Standard)

3-to-1 Vendor recipe

When using the Vendor recipe, it is important to use the same Scarab as input for the trades to not scew the result. When using different scarabs as inputs, separate the results and treat them as different Datasets:

  1. Choose Your Input Scarab: Select a specific scarab type to use as input (e.g., Abyss Scarab, Ambush Scarab)
  2. Vendor Recipe: Sell 3 scarabs of that type to a vendor to receive 1 random scarab back
  3. Batch Collection:
    • Since you can only fit 60 items in your inventory, and for efficiency, work in batches of 180 scarabs (3 batches of 60)
    • Sell 180 input scarabs (3 Ã— 60) to get 60 random scarabs back
    • Repeat this process as many times as you can
  4. Record Results: Note exactly which scarab you used as input and all the scarabs you received back, including their counts
  5. Continue Collecting: The more batches you complete, the more accurate your dataset will be

Opening veiled Scarabs

Veiled Scarabs are only available in Standard. When opening veiled Scarabs, you will receive a random Scarab.

Simply opening the veiled Scarabs and recording the Scarab you received is sufficient to collect data on the weight of the Scarabs.
