Contributing Scarab Datasets

Overview

To contribute useful scarab datasets, you'll need to collect data using the 3-to-1 vendor recipe: sell 3 scarabs of the same type to a vendor to receive 1 random scarab back. This recipe allows you to systematically collect data about scarab conversion rates.

Data Collection Approach

The correct approach for collecting scarab data is to use the 3-to-1 Vendor recipe. It is crucial to use the same Scarab as input for the trades to not scew the result. When using different scarabs as inputs, separate their resultings and treat them as different Datasets:

  1. Choose Your Input Scarab: Select a specific scarab type to use as input (e.g., Abyss Scarab, Ambush Scarab)
  2. Vendor Recipe: Sell 3 scarabs of that type to a vendor to receive 1 random scarab back
  3. Batch Collection:
    • Since you can only fit 60 items in your inventory, and for efficiency, work in batches of 180 scarabs (3 batches of 60)
    • Sell 180 input scarabs (3 × 60) to get 60 random scarabs back
    • Repeat this process as many times as you can
  4. Record Results: Note exactly which scarab you used as input and all the scarabs you received back, including their counts
  5. Continue Collecting: The more batches you complete, the more accurate your dataset will be

Quality Guidelines

  • Sample Size: Aim for at least 1000+ input scarabs for basic results, 10000+ for better accuracy
  • Data Accuracy: Ensure all scarabs received from the vendor are accurately counted, you can setup a dedicated stashtab and use eg wealthyexile to support in the counting
  • Completeness: Include all scarab types that you received back, even if count is 0
  • Input Tracking: Clearly document which scarab type you used as input

Best Practices

  • Use the Vendor Recipe: Always use the 3-to-1 vendor recipe (sell 3 scarabs → get 1 back) with the same Scarab
  • Work in Batches: Process in batches of 180 (3 inventory loads of 60) for efficiency
  • Track Input Carefully: Clearly note which scarab type you're using as input
  • Count Accurately: Count all output scarabs
  • (Optinal) Multiple Sessions: Collect data over multiple sessions to build a larger sample size
